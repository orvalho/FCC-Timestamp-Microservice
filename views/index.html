Timestamp Microservice

Description

This service provides time in both unix and utc formats.

Simply add /api/timestamp/ and the time to the tail-end of this url!

Example API Calls

  • https://silken-epoch.glitch.me/api/timestamp/2015-12-25
  • https://silken-epoch.glitch.me/api/timestamp/1450137600000

Example Output

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
